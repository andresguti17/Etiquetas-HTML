    • INTRODUCCIÓN A HTML

    El lenguaje de marcado de hipertexto (HTML) es el lenguaje de marcado estándar para documentos diseñados para su visualización en navegadores web.

    Estructura básica de un documento HTML: 

    
        <!DOCTYPE html>
        <html>
        < head>
            <meta charset="UTF-8" >
            <meta name= "viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>
            <h1> Título</h1>
        </body>
        </html>

    Partes principales de un documento HTML

    • <!DOCTYPE html> → Indica al navegador que el documento usa HTML5.
    • <html> → Es la etiqueta raíz que contiene todo el documento.
    • <head> → Contiene información sobre la página como el título, los metadatos y enlaces a archivos CSS o JavaScript.
    • <title> → Define el título que aparece en la pestaña del navegador.
    • <body> → Contiene todo el contenido visible de la página.

    Buenas prácticas de HTML

    • Usar etiquetas semánticas como <header>, <nav>, <section>, <article>, <footer> para que el contenido tenga significado y sea más accesible.
    • Mantener una estructura ordenada y bien indentada para que el código sea fácil de leer.
    • Usar atributos adecuados, como alt en imágenes y label en formularios para mejorar accesibilidad.
    • Evitar estilos en línea y preferir archivos CSS externos.
    • Optimizar imágenes y recursos para mejorar el rendimiento.

    Atributos importantes de HTML

    • <id> Identifica un elemento de forma única en el documento.
    • <class> Asigna una o más clases a un elemento para aplicar estilos CSS.
    • <style> Permite aplicar estilos directamente a un elemento.
    • <src> Especifica la ruta de una imagen o archivo multimedia.
    • <alt> Proporciona texto alternativo para imágenes, útil para accesibilidad.
