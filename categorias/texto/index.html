ETIQUETAS

    • Texto en HTML

    Son etiquetas de encabezado que se usan para definir títulos y subtítulos en una página web. Van desde <h1> (más importante) hasta <h6> (menos importante).

    Sintaxis Basica

    
     <h1 class="titulo" id="principal"> Título principal</h1>
        <h2>Subtítulo</h2>
        <h3 style="color: blue;"> Sección</h3>
        <h4> Subsección</h4>
            <h5> Detalle</h5>
            <h6> Nota pequeña</h6>

    visualización

    Buenas practicas

    • Usar <h1> debe usarse solo una vez por página (título principal).
    • Mantener orden jerárquico (no saltar de <h1> a <h4> sin razón).
    • Usar para estructura, no para tamaño de texto.
    • No usar solo por diseño visual (eso es trabajo de CSS).
