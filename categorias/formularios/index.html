    • INTRODUCCIÓN A FORMULARIOS EN HTML

    En HTML, una lista es una estructura que permite organizar y mostrar elementos relacionados de forma ordenada o desordenada dentro de una página web.

    Actualmente, en HTML existen 6 etiquetas. Mediante estas 6 etiquetas se pueden mencionar 3 tipos de listas. Primero, mencionaremos todas las etiquetas para después mostrar los tipos de listas que se pueden hacer con estas.

    <form>

    Contenedor principal que agrupa todos los elementos de un formulario y permite enviar datos al servidor.

    
    <form>

    </form>

    <label>

    Define una etiqueta descriptiva para un campo de entrada. Mejora la accesibilidad.

    
    <form>
        <label>Ingrese su nombre:</label>
    </form>

    <input>

    Campo de entrada de datos. Su comportamiento cambia según el atributo type.

    TIPOS DE INPUT

    text

    Campo para texto corto (nombre, usuario, etc.).

    
        <form>
            <label for="nombre">Ingrese su nombre:</label>
            <input type="text" id="nombre" name="nombre">
        </form>

    email

    Campo para correos electrónicos (valida formato automáticamente).

    
        <form>
            <label for="correo">Ingrese su correo electrónico:</label>
            <input type="email" id="correo" name="correo">
        </form>

    password

    Campo para contraseñas (oculta los caracteres).

    
        <form>
            <label for="contraseña">Ingrese su contraseña:</label>
            <input type="password" id="contraseña" name="contraseña">
        </form>

    number

    Permite ingresar solo números.

    
        <form>
            <label for="identificacion">Ingrese su documento de identificación:</label>
            <input type="number" id="identificacion" name="identificacion">
        </form>

    date

    Muestra un selector de fecha.

    
        <form>
            <label for="fecha">Ingrese su fecha de nacimiento:</label>
            <input type="date" id="fecha" name="fecha">
        </form>

    radio

    Permite seleccionar una sola opción dentro de un grupo.

    
        <p>Seleccione su estado:</p>
        <form>
            <input type="radio" id="casado" name="estado" value="casado">
            <label for="casado">Casado</label><br>
            <input type="radio" id="soltero" name="estado" value="soltero">
            <label for="soltero">Soltero</label>
        </form>

    checkbox

    Permite seleccionar una o varias opciones.

    
        <form>
            <input type="checkbox" id="pregunta" name="pregunta">
            <label for="pregunta">¿Es mayor de edad?:</label>
        </form>

    file

    Permite subir archivos desde el dispositivo.

    
        <form>
            <label for="documento">Suba su documento de identidad escaneado:</label>
            <input type="file" id="documento" name="documento">
        </form>

    Vista previa:

    submit

    Botón que envía el formulario.

    
        <form>
            <input type="submit" value="Enviar">
        </form>

    Vista previa:

    <textarea>

    Campo para escribir texto largo (como comentarios o mensajes).

    
        <textarea name="mensaje" rows="10" cols="30">

        </textarea>

    <select>

    Crea una lista desplegable de opciones.

    
        <select id="idioma" name="idioma">

        </select>

    <option>

    Define cada opción dentro de un <select>.

    
        <select id="idioma" name="idioma">
            <option value="español">Español</option>
            <option value="ingles">Inglés</option>
            <option value="frances">Francés</option>
        </select>

    <button>

    Crea un botón que puede ejecutar acciones (enviar formulario, cancelar, etc.).

    
        <button type="button" onclick="alert('Felicidades!')">Clickea</button>

    <fieldset>

    Agrupa elementos relacionados dentro de un formulario.

    
        <fieldset>
        
        </fieldset>

    <legend>

    Título descriptivo para un <fieldset>.

    
        <fieldset>
            <legend>Información</legend>
            <label for="nombre">Nombre completo:</label><br>
            <input type="text" id="nombre" name="nombre"><br>
        </fieldset>
