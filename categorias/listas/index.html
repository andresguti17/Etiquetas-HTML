    • INTRODUCCIÓN A LISTAS EN HTML

    En HTML, una lista es una estructura que permite organizar y mostrar elementos relacionados de forma ordenada o desordenada dentro de una página web.

    Actualmente, en HTML existen 6 etiquetas. Mediante estas 6 etiquetas se pueden mencionar 3 tipos de listas. Primero, mencionaremos todas las etiquetas para después mostrar los tipos de listas que se pueden hacer con estas.

    <ul>

    Con esta etiqueta los elementos se muestran desordenados, listándolos con una viñeta a su lado izquierdo.

    <ol>

    Con esta etiqueta los elementos se muestran de forma organizada, siendo ennumerados.

    <li>

    Con esta etiqueta se enlistan los items.

    Mientras se use alguna de las 2 etiquetas mencionadas anteriormente.

    <dl>

    Con esta etiqueta se muestra una lista de definición.

    <dt>

    Con esta etiqueta se muestra el término o palabra a definir.

    <dd>

    Con esta etiqueta se define el término.

    TIPOS DE LISTAS

    Lista desordenada

    Para conseguir una lista que se muestra de forma desorganizada, debes usar la siguiente combinación de etiquetas:

    
    <ul>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
        <li>Elemento 3</li>
    </ul>

    Y esto se muestra de la siguiente forma:

    • Elemento 1
    • Elemento 2
    • Elemento 3

    Lista ordenada

    Para conseguir una lista que se muestra de forma organizada, debes usar la siguiente combinación de etiquetas:

    
    <ol>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
        <li>Elemento 3</li>
    </ol>

    Y esto se muestra de la siguiente forma:

    1. Elemento 1
    2. Elemento 2
    3. Elemento 3

    Lista de definición

    Para conseguir una lista donde se van a definir términos, debes usar la siguiente combinación de etiquetas:

    
    <dl>
        <dt>Elemento 1</dt>
        <dd>Definición 1</dd>
        <dt>Elemento 2</dt>
        <dd>Definición 2</dd>
        <dt>Elemento 3</dt>
        <dd>Definición 3</dd>
    </dl>

    Y esto se muestra de la siguiente forma:

    Elemento 1
    Definición 1
    Elemento 2
    Definición 2
    Elemento 3
    Definición 3
