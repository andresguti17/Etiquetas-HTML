    • Tablas en HTML

    Las tablas HTML permiten a los desarrolladores web organizar y tabular los datos como estadísticas, comparaciones o inventarios, de manera que sea fácil de leer y comprender en filas y columnas. 

     
                <th>  // Encabezado de la tabla
                <thead>  // Agrupo el contenido de los encabezados de la tabla 
                <tr>  // Define las filas de la tabla 
                <td>  // Define cada ccelda de la tabla 
                <tbody>  // Agrupa los cuerpos de cada tabla

    Ejemplo en codigo

    Buenas practicas

    • Usar <table> solo para datos, no para diseño.
    • Encabezados claros con <th> y <code>scope</code> para accesibilidad.
    • Filas <tr> y celdas <td> bien organizadas.
    • Usar <caption> y <thead>/<tbody>/<tfoot> para estructura y claridad.
    • Estilos con CSS, no con atributos HTML, y evitar tablas anidadas innecesarias.
